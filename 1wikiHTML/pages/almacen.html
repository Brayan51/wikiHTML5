Definicion de HTML5

Es un lenguaje markup (Hyper Text Markup Language) usado para estructurar y presentar el contenido para la web. Es uno de los aspectos fundamentales para el funcionamiento de los sitios, pero no es el primero, siendo la quinta revision del estandar que fue creado en 1990. Html5 tenemos otras posibilidades para explotar usando menos recursos. Html5 es la ultmia version de HTML. Se trata de una nueva version de con nuevos elementos, atributos y comportamientos. Contiene un conjunto mas amplio de tecnologias que permite a los sitios web y a las aplicaciones ser mas diversas y de gran alcance.

Usos que puede ofrecer HTML5
Paginas web con elementos multimedia.
La posibilidad de poder incluir videos de presentacion de un producto, criticas en video, podcast, muestras de musica. Tambien puedes subir tus videos a paginas de terceros como Vimeo o Youtube e incrustarlos en tu nuevo sitio web.

Funciones de geolocalizacion.
Permite al sitio detectar la ubicacion de cada usuario que ingresa al sitio web. Esto puede tener diversos usos, por ejemplo ofrecer opcion de idiomas segun el lugar de ubicacion del usuario o para enlazarlo a la pagina oficial de la marca en el pais en el que se encuentra.
